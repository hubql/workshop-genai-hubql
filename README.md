# workshop-genai-hubql

### Prerequisites

To be able to run [supabase](https://supabase.com/) please make sure you have [Docker](https://www.docker.com/products/docker-desktop/) installed on your host machine.

Node version: 20.11.1

### Local Setup

```bash
cp .env.example .env
pnpm i
pnpx supabase start
pnpm run migrate
pnpm run dev
```
